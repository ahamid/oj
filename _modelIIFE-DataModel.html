Class: DataModel

DataModel

new DataModel(name, extend)

Private class representing the construction of a dataModel. It returns a n$.dataModels.dataModel instance with collections for adding columns and subscribers.

Parameters:
Name Type Description
name String

The ClassName of the dataModel to associate with ExtJS
extend String

[extend='Ext.model.Model'] An ExtJs class name to extend, usually the Model model
.dataTypeCollection n$.dataTypes.collection

[fields=new Array()] An array of fields to load with the dataModel. Fields can be a n$.dataTypes.collection or an array (e.g. ['name', 'string', 'Bob'])
